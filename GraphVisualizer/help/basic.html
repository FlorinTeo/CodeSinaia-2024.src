Graph Visualizer reference

Graph Visualizer allows modeling graph algorithms by creating and interacting with a visual representation of a generic Graph. Using a combination of keys and mouse clicks, moves, drags and wheel actions on a simple canvas, you can define and alter the structure the Graph and model algorithmic executions by highlighing its nodes or edges and by referencing individual nodes.

The main concepts used by the Graph Visualizer are as follows:
Nodes Edges References
The Nodes can be created, deleted or repositioned via Node basic operations. Nodes contain a label and a state which can be altered, via Node context menu options. The Edges are directional links between Nodes and can be created, reoriented or deleted via Edge basic operations. The References are pointers to individual Nodes, that can be created, deleted or repositioned via Reference basic operations. They can be renamed via a Reference context menu

Node basic operations top
Add a Node
On an open area on the canvas, press the [Ctrl] key and [left-click].
Remove a Node
Point to a Node, press the [Ctrl] key and [left-click]. Removing a Node removes all outgoing and incoming edges and clears the content of the auxiliary data structures.
Reposition a Node
Point to a Node and [drag] it to a new position on the canvas. The Node remains attached to the Graph and preserves all its properties.
Change the color of a Node
Point to a Node and [scroll down] the mouse wheel to loop through five predefined colors: [blue, red, green, magenta, yellow] or [scroll up] to reset the color of the Node to the default gray.
Change the label of a Node
Point to a Node, press the [Ctrl] key and [scroll up] or [scroll down] the mouse wheel. The label of the Node will loop through available letters in the range [1-9A-Z]. If no letter is available, the default '?' label will be used.
Node context menu top
Point to a Ndoe and [right-click]. The context menu that opens can be used for the following actions:
Customize the node's label
Hover over the edit area next to the [Label___] text. Type in the desired new label, then press <Enter> to set it into the Node.
Change the node's state
Hover over the edit area next to the [State___] text. Type in the desired new state for the Node, then press <Enter> to set it into the Node.

Edge basic operations top
Add/Remove a directional Edge
Press [Ctrl] key while pointing to a Node and [dragging] a line over to another Node. A directional Edge will connect the first Node to the second one, or, if the Edge existed already, it will be removed.
Add/Remove a bi-directional Edge
Press both [Ctrl] and [Shift] keys while pointing to a Node and [dragging] a line over to another Node. A bi-directional Edge will connect the first Node to the second one, or, if the Edge existed already, it will be removed.
Change the color of an Edge
Hover in proximity of an Edge and [scroll down] the mouse wheel to loop through five predefined colors: [blue, red, green, magenta, yellow] or [scroll up] to reset the color of the Edge to the default gray.

Reference basic operations top
Add/Remove a Reference to a Node
On an open area on the canvas, press [Shift] and [left-click]. A Reference label will be created, or if one exists already, it will be removed. If a the Reference is created in the proximity of a Node, it will automatically attach itself to that Node. Otherwise it wll be created as an independent, (null) reference.
Reposition a Reference
Point to the middle area of the Reference and [drag] it to a new position. The Reference will automatically detach / reattach to the Node in its closest proximity. If no Node is close enough, it will will become a (null) reference.
Reference context menu top
Renaming a Reference
Point to the middle of a Reference and [right-click]. In the context menu that opens up, hover over the edit area next to the [Label___] text. This will set the input focus on the edit field. Type in the desired new name for the Reference, followed by the <Enter> key to set the new name for the Reference. Press the <Escape> key or click anywhere on the canvas to cancel the operation and dismiss the menu.
